# OlalaChess Masterclass

GM Arthur Kogan's OlalaChess — premium chess training, coaching, and positive thinking methodology.

## Tech stack

- Vite + React + TypeScript
- shadcn/ui
- Tailwind CSS

## Getting started

```sh
git clone https://github.com/carolina-kp/olalachess-masterclass
cd olalachess-masterclass
npm install
npm run dev
```
